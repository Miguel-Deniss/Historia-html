História do HTML

Quem criou o HTML?

O HTML foi criado por Tim Berners-Lee, um cientista da computação britânico. Ele também é conhecido como o criador da World Wide Web. Seu trabalho foi muito importante para o desenvolvimento da internet, pois permitiu a criação de páginas ligadas por links.

O que é HTML?

HTML significa Linguagem de Marcação de Hipertexto. Ele é usado para criar e organizar páginas da web, colocando textos, imagens, links, listas, vídeos, tabelas e formulários.

Principais versões do HTML

  • 1991 — Criação do HTML
  • 1995 — HTML 2.0
  • 1997 — HTML 3.2
  • 1999 — HTML 4.01
  • 2000 — XHTML
  • 2014 — HTML5
